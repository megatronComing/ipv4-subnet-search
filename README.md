# ipv4-subnet-search
search for minimum subnets which can accommodate give number os host machines
Usage: find_subnet.py IP_ADDRESS/SUBNET_MASK_LEN HOST1 HOST2 HOST3
Required libraries: ipaddress, tabulate
Tested under:
    macos Monterey 12.7
    python 3.11.3
